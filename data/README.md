# Exercise data (CC0)

Everything in this folder is released into the public domain under CC0 (see `LICENSE`).
Reuse it anywhere, including in other apps — no attribution required (appreciated, not required).

- `exercises.json` — the seed library. Add to it following `schema.json`.
- `patterns.json` — movement-pattern archetypes; each points at a figure.
- `schema.json` — JSON Schema for a single exercise entry.

After editing, run `npm run validate` (or `node scripts/validate-data.mjs`).
